# 챗 서비스 사용자 매뉴얼

**챗 서비스**(`app-internal`)는 사내 임직원용 채팅 서비스입니다. 클로드·ChatGPT처럼 채팅창에 질문을 입력하면, 사내 문서·데이터베이스·웹 검색 결과를 근거로 답변을 받을 수 있습니다.

> 이 문서는 **일반 사용자 화면**을 기준으로 합니다. 관리자 화면은 [어드민 매뉴얼](admin-manual.md)을 참고하세요.

---

## 1. 로그인

1. 접속 주소로 이동하면 로그인 화면이 나타납니다.
2. **이메일**, **비밀번호**를 입력하고 **로그인** 버튼을 누릅니다.
3. 관리자가 발급한 **초기 비밀번호**로 처음 로그인하면 "비밀번호 변경 필요" 모달이 자동으로 뜹니다. **새 비밀번호**와 **새 비밀번호 확인**을 입력하고 **비밀번호 변경**을 눌러야 서비스를 이용할 수 있습니다.

---

## 2. 화면 구성

채팅 화면은 좌측 사이드바(대화 목록), 중앙 대화창, 우측 파라미터 패널(열었을 때만 표시)로 구성됩니다.

- 좌측 상단 **▾** 아이콘: 사이드바 접기/펼치기
- 중앙 상단: 현재 대화방 제목 (클릭하면 이름 편집 가능)
- 우측 상단 아이콘 3개
  - 🔑(열쇠) — 비밀번호 변경 화면으로 이동
  - ⚙(톱니바퀴) — 파라미터 설정 패널 열기/닫기
  - ⏻(로그아웃) — 로그아웃

---

## 3. 대화 관리 (좌측 사이드바)

- **새 대화** 버튼 — 새 대화방 생성
- 대화 목록의 각 항목 우측 **⋯(더보기)** 버튼을 누르면
  - **이름 변경** — 대화 제목을 직접 수정
  - **상단 고정 / 고정 해제** — 목록 맨 위에 고정 (핀 아이콘 표시)
  - **삭제** — 대화 삭제 (별도 확인창 없이 즉시 삭제되므로 주의)
- 대화 제목은 첫 질문을 보낸 뒤 LLM이 자동으로 요약해 붙여줍니다(실패 시 기본 제목 유지). 언제든 수동으로 다시 바꿀 수 있습니다.
- 대화가 응답을 기다리는 중이면 목록에 로딩 스피너가 표시됩니다.

---

## 4. 메시지 보내기

- 입력창에 텍스트 입력 후 **Enter**로 전송, **Shift+Enter**로 줄바꿈.
- **↑ / ↓** 방향키 — 이전에 보낸 메시지 히스토리를 다시 불러옵니다(입력창이 비어 있거나 히스토리 탐색 중일 때).
- 전송 버튼(➤)은 텍스트나 첨부파일/이미지가 하나라도 있어야 활성화됩니다.

### 4-1. 슬래시 커맨드 (라우팅 강제)

입력창에 `/`만 입력하면 커맨드 선택 메뉴가 뜹니다. `↑↓`로 이동, `Enter`/`Tab`으로 선택, `Esc`로 취소할 수 있습니다.

| 커맨드 | 동작 |
|--------|------|
| `/rag ` | 내부 문서 검색 우선 실행 (결과 없으면 웹 검색으로 폴백) |
| `/web ` | 웹 검색 우선 실행 (거부되면 내부 문서 검색으로 폴백) |
| `/sql ` | SQL 직접 실행 (의도 분류기를 건너뜀, 폴백 없음) |

커맨드를 안 쓰면 시스템이 질문 내용을 보고 자동으로 RAG/SQL/HYBRID/웹검색 중 하나를 선택합니다.

### 4-2. 파일 첨부

- 📎(클립) 아이콘으로 파일 선택. 지원 확장자: `.pdf .docx .doc .pptx .ppt .xlsx .xls .txt .md .csv` (여러 개 동시 첨부 가능)
- 첨부한 파일은 입력창 위에 칩(chip) 형태로 표시되며 **×**로 개별 삭제 가능
- 전송 시 파일이 먼저 업로드되고, 일부가 실패하면 "업로드 실패: 파일명" 배너가 표시됩니다. 텍스트도 없고 성공한 파일도 없으면 전송 자체가 취소됩니다.

### 4-3. 이미지 첨부

- 🖼(이미지) 아이콘으로 이미지 선택 (여러 장 가능)
- **HEIC/HEIF**(아이폰 사진) 파일은 자동으로 JPEG로 변환되어 첨부됩니다. 변환 실패 시 해당 파일만 건너뜁니다.
- 이미지만 보내면 비전 모델이 이미지를 직접 분석해 답변하고, 텍스트 질문과 함께 보내면 이미지 분석 + 내부 검색을 결합해 답변합니다.

---

## 5. 답변 화면 읽는 법

- 답변 상단의 배지로 어떤 경로로 처리됐는지 알 수 있습니다: `📄 RAG` · `📊 SQL` · `🔀 HYBRID` · `🖼️ IMAGE` · `📎 FILE` · `🌐 WEB`
- 답변은 마크다운으로 렌더링되며 표(가로 스크롤 지원)도 정상적으로 표시됩니다.
- **생성된 SQL** — SQL 질의였을 경우 실제 실행된 쿼리문이 코드 블록으로 표시됩니다.
- **CSV 다운로드** 버튼 — SQL 조회 결과가 많을 때 답변 안에 다운로드 버튼이 자동으로 삽입됩니다.
- **검색 출처** — 웹 검색 결과일 경우 참고한 URL 목록이 표시됩니다.
- **출처 N개** — 내부 문서 검색(RAG) 결과일 경우 클릭하면 펼쳐지며, 각 출처의 제목·관련도(%)·소스가 표시됩니다.
- 답변 우측 하단에 응답 소요 시간이 표시되고, 답변을 기다리는 동안에도 실시간으로 경과 시간이 올라갑니다.

---

## 6. 파라미터 설정 (우측 패널)

우측 상단 톱니바퀴(⚙) 아이콘을 누르면 파라미터 패널이 열립니다. 대화별로 아래 값을 조정할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 시스템 프롬프트 | 답변 스타일·지침을 직접 지정 |
| Top-K (1~20) | 검색해 올 문서 조각 개수 |
| 유사도 임계값 (0~1) | 이 값보다 유사도가 낮은 문서는 제외 |
| Temperature (0~2) | 답변의 창의성/무작위성 |
| Top P (0~1) | 답변 생성 시 확률 샘플링 범위 |
| 최대 토큰 (100~4096) | 답변 최대 길이 |
| 쿼리 타임아웃(초) (5~60) | SQL 조회 제한 시간 |
| 최대 결과 행수 (10~10000) | SQL 조회 결과 최대 행수 |
| 라우팅 강제 | AUTO / FORCE_RAG / FORCE_SQL / FORCE_HYBRID |
| 하이브리드 스타일 | BALANCED / SQL_FIRST / RAG_FIRST |
| 최대 히스토리 턴수 (1~50) | LLM에 전달할 이전 대화 턴 수 |

- 회색으로 잠긴(🔒) 항목(SQL Temperature, Few-Shot 예시 수, 최대 컨텍스트 토큰)은 사용자가 바꿀 수 없고 관리자만 조정 가능합니다.
- 각 파라미터는 관리자가 설정한 허용 범위(최솟값~최댓값) 안에서만 조정됩니다.
- 하단 **초기화** 버튼으로 기본값으로 되돌릴 수 있습니다.

---

## 7. 비밀번호 변경

우측 상단 🔑 아이콘 또는 `/settings` 화면에서 **현재 비밀번호**, **새 비밀번호**(8자 이상), **새 비밀번호 확인**을 입력해 변경합니다.

---

## 8. 로그아웃

우측 상단 ⏻ 아이콘을 누르면 로그아웃되어 로그인 화면으로 이동합니다.

---

## 9. 관리자 메뉴

계정 역할이 `ADMIN` 또는 `SUPER_ADMIN`이면 좌측 사이드바 하단에 **관리자 설정** 메뉴가 표시됩니다. 상세 내용은 [어드민 매뉴얼](admin-manual.md)을 참고하세요.

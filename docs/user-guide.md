# RAG 서비스 사용자 가이드

> 버전: Phase 0 | 최종 수정: 2026-05-28

## 시작하기

### 접속 방법

1. 브라우저에서 `https://[회사도메인].ragservice.com` 접속
2. 관리자로부터 발급받은 이메일·비밀번호로 로그인

### 첫 번째 질문하기

1. 화면 하단 입력창에 질문을 입력합니다
2. Enter 키 또는 전송 버튼을 클릭합니다
3. AI가 회사 내부 자료를 검색해 답변합니다 (10~60초 소요)

---

## 질문 유형

| 질문 유형 | 예시 | 설명 |
|----------|------|------|
| 문서 검색 (RAG) | "보안 정책 문서에서 원격 근무 관련 내용 알려줘" | 등록된 내부 문서 기반 답변 |
| 데이터 조회 (SQL) | "지난달 부서별 매출 알려줘" | 회사 데이터베이스 수치 조회 |
| 혼합 (Hybrid) | "고객 수와 보증 정책 함께 알려줘" | 문서 + 데이터 통합 답변 |
| URL 분석 | "https://... 이 페이지 요약해줘" | 외부 웹페이지 내용 분석 |
| 파일 분석 | PDF/Word/Excel 첨부 후 질문 | 첨부 파일 내용 기반 답변 |
| 이미지 분석 | 이미지 첨부 후 "이 이미지 설명해줘" | 이미지 내용 분석 |

### 자동 분류

질문 형태에 따라 적절한 경로가 자동 선택됩니다. 답변 상단에 경로 라벨(RAG, SQL 등)이 표시됩니다.

---

## 파일 첨부

### 지원 형식

PDF, Word(.docx), PowerPoint(.pptx), Excel(.xlsx), 텍스트(.txt, .md, .csv)

### 제한

- 단일 파일: **최대 30MB**
- 누적 크기: **최대 100MB**
- 파일은 **24시간 후 자동 삭제**됩니다

### 방법

1. 입력창 왼쪽 첨부 아이콘 클릭
2. 파일 선택 (여러 파일 동시 첨부 가능)
3. 미리보기 카드 확인 후 질문 입력

---

## RAG 파라미터 조정

화면 우측 상단 RAG 파라미터 버튼으로 검색 설정을 조정할 수 있습니다.

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| Top-K | 5 | 검색 문서 수 (많을수록 넓은 검색) |
| 유사도 임계값 | 0.65 | 낮을수록 더 많은 문서 포함 |
| Temperature | 0.3 | 높을수록 창의적 답변 |
| 최대 토큰 | 2000 | 답변 최대 길이 |

> 파라미터 변경 후 이전 답변과 비교하려면 "이 답변 다시 받기" 토스트를 활용하세요.

## 출처 확인

답변 하단 **출처 N개** 버튼을 클릭하면 답변 근거 문서를 확인할 수 있습니다.
답변 본문의 **[1]**, **[2]** 는 해당 출처 번호입니다.

## 피드백

답변 하단 좋아요/싫어요 버튼으로 답변 품질을 평가해 주세요.
싫어요 선택 시 사유 선택 가능: **정확하지 않음 / 부족함 / 무관함 / 기타**

## 문제 해결

| 증상 | 해결 방법 |
|------|----------|
| 답변이 너무 오래 걸림 | 최대 10분 대기 (LLM 처리 시간). 그 이상이면 새로고침 후 재시도 |
| "관련 정보를 찾을 수 없습니다" | 다른 표현으로 재시도. 관련 문서가 등록되지 않았을 수 있습니다 |
| "보안 정책 위반" 메시지 | 개인정보·기밀 데이터 질문 제한. 관리자 문의 |
| 응답이 중단됨 | 상단 "다시 시도" 버튼 클릭 |
| 파일 업로드 실패 | 파일 크기(30MB)·형식 확인 |

시스템 장애 여부: [상태 페이지](/status) 확인
